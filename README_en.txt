###### Spindle (Research Package) version 1.1.0 stable 27/04/2015

####General Usage Notes  
Spindle is a MySql/PHP Open Source application designed for the development of Alternate Reality Games (ARGs) based on Networked Narrative Storytelling (NNS) as part of the research project Principles of Liberty (PoL) conducted by Tabea Hirzel at the SMC University, Zug, Switzerland (http://smcuniversity.com).

    + NO support of any kind is provided.
    + NO reliability for any harms to your system are given.
    + Please, install under your own risk!!!

#### Package  
This package includes:
    + Standalone version 1.1.0 stable
    + Server based version 1.1.0 stable
    + Installable Sql database
    + Uploadable Sql database with sample files
    + Documentation
    + User guide

#### Includes  
This version includes the following functionalities:
    + Agents 
      ++"user"-"agents" integration (manual only)
    + Bibliography
      ++ document upload (enabled)
      ++ transcript upload (enabled)
    + Encounters
      ++ NodeXL download (enabled)
    + Storylines
      ++ Page2Stage compatible (ok)
    + Dramatica 
      ++ theme relationships (partially automatized

#### System Requirements (Standalone):
    + 128 MB RAM
    + Webbrowser

#### System Requirements (for local installation):
    + 128 MB RAM
    + 192 MB free fixed disk
    + Windows 2000, XP (Server 2003), Vista (Server 2008), 7
    + Server environment (XAMPP, LAMP, or similar)
    + all systems 32 bit (64 bit should also work)
